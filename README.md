# Potential Backbone
Compute a scalar potential field within a 3D voxel grid tunnel, compute trajectory of saddle points and output the backbone of the tunnel.

## Prerequisites

Eigen3 library
```
sudo apt install libeigen3-dev
```

## Get Started
In the root directory
```
cd build
cmake ..
make
```
Then run the code
```
.\potential
```
